# Redaktə Hesabatı — «Hikmətli hekayələr»

Mənbə sənəd: `documents/Hikmətli hekayələr (bx-kt).docx`
Redaktə olunmuş fayllar: `documents/edited_stories/<Hekayə adı>_edited.docx`

Dil: müasir ədəbi Azərbaycan dili. Redaktə yalnız Azərbaycan dili normaları əsasında aparılıb; başqa dillərin qaydaları tətbiq olunmayıb.

Redaktənin prinsipi: orijinalın **mənası, tonu, atmosferi və müəllif üslubu** qorunmaqla orfoqrafiya, qrammatika, durğu işarələri, cümlə quruluşu və üslub qüsurları düzəldilib. Hekayələr yenidən yazılmayıb — yalnız cilalanıb.

Ümumi tətbiq olunan dəyişikliklər:
- Dırnaq işarələri Azərbaycan standartına uyğun «...» şəklinə salınıb.
- Hekayə ilə ondan çıxarılan nəticə arasındakı `-----` ayırıcısı `* * *` ilə əvəzlənib.
- Dialoq tire (—) və durğu işarələri nizamlanıb.
- Təkrar boşluqlar, boşluq/durğu işarəsi qüsurları aradan qaldırılıb.

Fayl adı formatı: `Hekayə adı_edited.docx`.

Cəmi hekayə: **252**. (Boş və ya təkrar başlıqlar aşkarlandıqda hesabatda qeyd olunur.)

## Toplu 1 (hekayə 1–15)

### 1. Adam olmaq
- Sadalama cümlələri vahid quruluşa salındı (hər bənd nöqtəli vergüllə, sonuncu nöqtə ilə); «Adam olmaq:» girişindən sonra kiçik hərflə davam etdirildi.
- «nə kimi meyarlar» → «hansı meyarlar» (müasir norma).
- «oğru olmayan» → «oğurluq etməyən», «Yalançı olmayan» → «yalan danışmayan» (üslubi yeknəsəklik).
- Nəticə cümləsi aydınlaşdırıldı: «deməli, sevgisiz bir cəmiyyət var» → «deməli, cəmiyyət də sevgisizdir».
- Dırnaqlar «...» formatına salındı.

### 2. Ağızlara söz, ayaqlara toz olma
- İzahedici cümlə səlisləşdirildi: «ayağın altındakı toz kimi — heç kəs...» → «...toz kimi olmaqdır; heç kəs...».
- «tökülər, unudular» → «tökülüb unudular»; dırnaqlar nizamlandı.

### 3. Ağzını açma
- «gözəl bir şey söyləmək» → «gözəl bir söz söyləmək» (kontekstə uyğun).
- Sadalamada vergüllər nöqtəli vergüllə əvəz olundu, oxunaqlıq artırıldı.

### 4. Ailənizin dəyəri
- Danışan adları vahidləşdirildi: «ATA:»/«Ata:» → hamısı «Ata:»; hər replika dialoq tiresi (—) ilə verildi.
- İki qırıq replika («Ah!» və borc istəyi) məntiqə uyğun birləşdirildi.
- «almağın üçün... xərcləməyin üçün» → «almaq... xərcləmək üçün»; «bağladı» → «örtdü»; daxili düşüncə «...» dırnağına alındı.
- Nəqli hissədə artıq sözlər çıxarıldı, ahəng qorundu.

### 5. Alaq otlarını kökündən qoparmaq lazımdır
- «istəyərdi» → «istəyirdi»; tire ilə verilən əlavələr nöqtəli vergül/iki nöqtə ilə nizamlandı.
- Atanın sözləri dırnağa alındı; nəticə cümləsində uzunluq azaldılıb aydınlaşdırıldı.

### 6. Albrecht Dürer
- «atası» → «ata», «yaşayırdılar» → «yaşayırdı» (mübtəda-xəbər uzlaşması).
- «qalib gəldi» → «qazandı», «yardım edəcək» → «maddi dayaq olacaqdı», «göz yaşlarına qərq oldu» → «göz yaşlarına boğuldu».
- Xüsusi adlar və əsər adı «Dua edən əllər», «Albertina» dırnağa alındı; tarix diapazonu saxlanıldı.

### 7. Allahın dostu
- **Məna qüsuru düzəldildi:** «bir uşağın ayaqqabısız olmadığını görür» (əks məna) → «ayaqqabısı olmayan bir uşaq görür».
- «ALLAH», «ALLAHA», «ALLAHIN» böyük-hərf yazılışı mətn içində «Allah», «Allaha», «Allahın» kimi normallaşdırıldı.
- Sonuncu replikadakı artıq «deyir» və qırıq dırnaq düzəldildi.

### 8. Allahın rizası
- «Hacə Ubeydullah Əhrar:» → «...deyir:» (giriş tamamlandı); sitat vahid «...» içinə alındı.
- «yalan, oğurluqdan əvvəl» → «yalandan, oğurluqdan əvvəl» (hal uzlaşması).

### 9. Allahın verdiyi nemətlər
- Başlıqdakı «ALLAHNIN» → «Allahın» (orfoqrafiya).
- Sadalama vahid formata salındı (kiçik hərf + nöqtəli vergül); «necə yetişdirdiyinə» → «onu necə yetişdirdiyinə».
- «kimliğini» → «kimliyini» (orfoqrafiya).

### 10. Altmışdan sonra nə dəyişdi
- Orfoqrafiya: «gənç» → «gənc», «güçlü» → «güclü», «təqaudçü» → «təqaüdçü», «meyarlarlı» → «meyarları».
- **Məna/söz seçimi:** «ifadələri isə əsla unutdum» (məntiqsiz) → «...heç işlətmirəm»; «gerçəkdən» → «həqiqətən».
- Cümlələr qısaldılıb səlisləşdirildi; təkrarlanan «Nə də olsa» ahəngi qəsdən saxlanıldı (müəllif üslubu).

### 11. Altmışı haqlamış olanlar üçün...
- **Qrammatika:** «sevginizi həmişə yaşadın» → «...yaşayın» (əmr forması).
- «pozitiv enerji» → «müsbət enerji»; «Bərbərinizin yanına getməyi» → «Bərbərə getməyi».
- Uzun cümlələr bölündü; «Kimsə deyib ki...» sitatı səlisləşdirildi.

### 12. Ana sevgisi
- Müəllif adı transliterasiyası: «Erich Fromm» → «Erix Fromm».
- «30 yaşa gəldim» → «30 yaşa çatdım», «50-yə yetişdim» → «50-yə çatdım».
- «ey sevimli ana... Ey gözəl ana...» — poetik sətir bölgüsü və atmosfer qorundu; dırnaqlar «...» formatına salındı.
- «qoxula» → «qoxusunu çək» (aydınlıq).

### 13. Artıq fərqindəyəm
- «bir bəhanə tapa biləcəklərinin» → «...biləcəyinin» (uzlaşma).
- İdiomatik «bir "əmma" tapdığının» → «bir «amma»sının olduğunun»; «sözünə sahib olmayan» → «sözünə əməl etməyən».
- «kimin yanında "özüm" isəm» → «...«özüm» oluramsa» (qrammatika); dırnaqlar nizamlandı.

### 14. Ana və oğul
- Bütün dialoq və daxili nitq «...» formatına salındı; tire ilə verilən replikalar nizamlandı.
- «şap-şaplarını» → «şəppişlərini»; «yetəcəkmi» → «yetərmi»; «çox bir əşyam yoxdur» → «çox əşyam yoxdur».
- «göz yaşlarına qərq oldu» tipli təkrar ifadələr sadələşdirildi; atmosfer və kövrək ton qorundu.
- Nəticədə «təkcə bədən olaraq deyil» → «təkcə cismən deyil»; «onu eşitməmək üçün» → «onu eşitmək üçün» (məntiqi düzəliş).

### 15. Aslanın izi
- Zaman vahidləşdirildi (keçmiş → indiki nəqli): «düşərək gəlib... çatar və... sual edər» → «düşüb gəlib... çatır və... soruşur».
- «istərsən» → «istəsən»; nəticədə söz sırası aydınlaşdırıldı, «iz»/«öz» dırnaqları saxlanıldı.

## Toplu 2 (hekayə 16–30)

### 16. Ata və oğul
- Bütün dialoq və nəqli hissədə dırnaqlar «...» formatına salındı, replikalar tire ilə nizamlandı.
- «acıq edərək» → «acıqlanıb»; «Madam» → «A kişi» (kontekstə uyğun müraciət); «Nə zamandan bəri» → «Nə vaxtdan».
- Təkrar və yöndəmsiz ifadələr sadələşdirildi; nəticədə «göstərmirlər» → «büruzə vermirlər», «getmək lazımdır» → «addım atmaq lazımdır».

### 17. Atanın izləri
- Dırnaqlar nizamlandı; nəticədə «təqlid edilən» → «təqlid olunan», söz sırası aydınlaşdırıldı.

### 18. Atların dostluğu
- Zaman və ifadələr səlisləşdirildi: «bəslənir» → «saxlanılır», «canına qıymamaqdan ötrü» → «ona qıymayıb».
- Uzun cümlələr bölündü; «digərlərinə yollarını tapmaqda» → «başqalarına yol tapmaqda».

### 19. Axtarmaq
- İki cümlə bir axara salındı (tire ilə); sondakı artıq təbboşluq/tab simvolu təmizləndi.

### 20. Az yeyən və düz gedən
- **Paralellik düzəldildi:** «Ən böyük qorunma... deyil» → «Ən yaxşı qorunma... yox»; «müalicə etmək deyil» / «vəkil tutmaq deyil» → «...yox» quruluşu vahidləşdirildi.

### 21. Baba və nəvəsi bazarlıqda
- Uzun, üst-üstə düşən cümlələr bölündü; birbaşa nitq «...» formatına salındı.
- «bağıraraq-çığıraraq» → «çığırıb»; «əməkdaşa» → «işçiyə»; zaman uzlaşdırıldı.

### 22. Bağ və bağban
- Dialoq nizamlandı; «ancaq» → «amma» (təkrarın qarşısı); nəticə səlisləşdirildi.

### 23. Bal ya quru çörək
- Mətnə qarışmış `-----` ayırıcısı düzgün ayrıldı və `* * *` edildi; tire ilə verilən əlavələr vergüllə nizamlandı.

### 24. Balıq dükanında
- «ALLAH», «ALLAHIM» → «Allah», «Allahım» (mətn içində orfoqrafiya).
- Dialoqlar tire və «...» ilə nizamlandı; «istərsən» → «istəsən», «bir quruş» → «bir qəpik», «yarım saatlıq» → «yarımsaatlıq».
- Uzun cümlələr bölündü, təkrarlar azaldıldı; emosional ton qorundu.

### 25. Baqqal dükanı və supermarket
- Yapışıq söz «qadınındükanı» → «onun dükanı»; elanlar «...» formatına salındı.
- «bu işi gördüm dedi» tipli yöndəmsiz quruluş düzəldildi: «Qoy camaat barı ucuz alsın.»

### 26. Baş mühasib və baş direktor
- Giriş tamamlandı («...deyir:»); suallar səlisləşdirildi; nəticə cümləsi aydınlaşdırıldı.

### 27. Başqalarının həyatına qarışmaq
- «ALLAH» → «Allah»; nəticə dırnaqları nizamlandı.

### 28. Bayrama hədiyyə
- Dialoq forması aydınlaşdırıldı (müraciət tire ilə); «Ala bilirsənsə» → «bacarırsansa».

### 29. Bənövşəyi çəpkən
- Mətnə düşmüş `**` və `**-**` işarə qalıqları təmizləndi; «Çünkü» → «Çünki».
- **Orfoqrafiya:** «ovladlarımıza» → «övladlarımıza»; uzun cümlələr bölündü, rəvayət tonu qorundu.

### 30. Bildiklərini söylə
- Cümlə sərhədləri bərpa olundu (nöqtələr əlavə edildi); yapışıq cümlələr ayrıldı.
- «çox böyük görmək» → «həddindən artıq böyütmək» (aydınlıq).

## Toplu 3 (hekayə 31–45)

### 31. Bilən və bilməyən
- Böyük hərflə vurğulanmış «ONA ÖYRƏDİN», «ONU OYADIN» və s. cümlə əvvəli normasına salındı: «Ona öyrədin!» və s.
- **Qeyd:** orijinalda nəticə yerində yalnız «?» durduğundan (mətn yoxdur) süni nəticə əlavə edilmədi; ayırıcı və boş nəticə çıxarıldı.

### 32. Bir cüt çəkmə
- «püşkatma» → «püşk», «imkansız ailə» → «kasıb ailə»; dialoq və dırnaqlar nizamlandı.
- **Qeyd:** sondakı «----- ?» (nəticəsi yazılmamış) çıxarıldı; süni nəticə uydurulmadı.

### 33. Bir ovuc ləbləbi
- Dialoqlar «...» və tire ilə nizamlandı; «istədin ki» → «istədin»; nəticə səlisləşdirildi.

### 34. Bir stəkan süd
- Anlaşılan sitat yenidən quruldu: «yaxşılıq edəndə əvəzsiz edin» → «yaxşılığı əvəzsiz edin».
- İmza sətri «...» formatına salındı; uzun cümlələr bölündü; «məmnun» tonu qorundu.

### 35. Bir tək anam ağlar
- «gerisi yalan ağlar» → «gerisi yalandan ağlar» (məna dəqiqləşdirildi).

### 36. Biz zənginik
- «sobaların yandığına» → «ocaqların yandığına»; sual «zənginlik mi» → «zənginlik»; cümlələr yığcamlaşdırıldı.

### 37. Bizdə hamı eynidir
- «Bölgə bələdiyyəsi» → «Rayon bələdiyyəsi»; böyük hərfli «EYNİ CÜR», «BİZDƏ HAMİ EYNİDİR» normal yazılışa salındı və orfoqrafiya düzəldildi («HAMİ» → «hamı»).
- Dialoqlar «...» formatına salındı, uzun cümlələr bölündü.

### 38. Boş vaxt
- «çünkü» → «çünki»; ifadələr cilalandı.

### 39. Boşunadır
- Böyük hərfli «BOŞUNADIR» cümlə içində normal yazıldı; «cahil ilə» → «cahillə».

### 40. Bu millət ilə dünya fəth edilə bilər
- Yer adı «Edirnə» → «Ədirnə» (Azərbaycan dilində qəbul olunmuş forma); «ALLAHA» → «Allaha».
- Dialoqlar nizamlandı, uzun cümlələr bölündü; tarixi rəvayət tonu qorundu.

### 41. Bu qayıq da boşdur
- Sətir sonlarındakı artıq boşluq/yeni sətir simvolları təmizləndi; «yiyəsini» → «sahibini»; cümlələr axıcı hala salındı.

### 42. Bu qədər az verməzdi
- Zaman vahidləşdirildi (indiki nəqli); dialoq və dırnaqlar nizamlandı.

### 43. Bu suallardan çəkinin
- Sadalama vahid sintaktik quruluşa salındı (kiçik hərf + vergül, sonda tire ilə ümumiləşdirmə).
- «yerli —yersiz» → «yerli-yersiz» (defis düzəlişi); dırnaqlar «...» formatına salındı.

### 44. Bu yol haraya gedir?
- «Heç bir yerə» → «Heç yerə» (təbii danışıq); «Gedib-gələnlər insanlardır» → «Gedib-gələn insanlardır».

### 45. Bunları etməyin
- «sağlığınızdan» → «sağlamlığınızdan», «məhkumun» → «məhbusun», «çünkü» → «çünki»; nəticə cümləsi cilalandı.

## Toplu 4 (hekayə 46–60)

### 46. Ciyər
- Uzun sadalama cümləsi vahid quruluşa salındı; ifadələr «...» içinə alındı; buraxılmış nöqtə əlavə olundu.
- «təsadüfi inkişaf etmir» → «təsadüfən inkişaf etmir».

### 47. Çalışın belə düşünün
- Bütün replikalar «...» formatına salındı; «İfadəlıərini» → «İfadəsini» (orfoqrafiya).
- Nəticə cümlə zənciri nöqtəli vergüllə nizamlandı.

### 48. Çatdırmadım, bacarmadım deməyin
- «məsud, kimiləri isə hüznlü bitirir gününü» → «Kimi gününü məsud, kimi hüznlü bitirir» (söz sırası); dırnaqlar nizamlandı.

### 49. Çətinliyi fürsətə çevirmək
- «çevirə bilməyi bacarmağımızdır» → «çevirə bilməkdir» (yığcamlıq); nəticə paralelliyi gücləndirildi.

### 50. Çobanı səsləmək lazımdır
- Dialoqlar tire ilə nizamlandı; «xeyir duası» → «xeyir-dua»; «çobana aid bir sürü quduz it» → «bir çobanın bir sürü quduz iti».
- «qaxılıb yerlərində duracaqlar» → «yerlərində quruyub qalacaqlar».

### 51. Çürük toxum
- Səhvən iki hissəyə bölünmüş replika birləşdirildi; «vücudunu» → «bədənini»; dırnaqlar «...» formatına salındı.

### 52. Daha effektiv yol
- Dialoq nizamlandı; «Buna mənim vaxtım haradandır?» → «Buna haradan vaxtım olsun?»; «effektiv» → «səmərəli».

### 53. Dedi-qodu
- Daxili düşüncə və dialoqlar «...» formatına salındı; «geri alıram» → «geri götürürəm»; «topa bilmədim» → «yığa bilmədim».

### 54. Dərdi izah etmək
- Sıralama vahid formata salındı; hər bənd «...» ilə nizamlandı; «daxilən yaşayır» → «için-için yaşayır» (üslub).

### 55. Dərviş əbası
- Dialoqlar tire və «...» ilə nizamlandı; «sordu» → «soruşdu»; uzun cümlələr yığcamlaşdırıldı; rəvayət tonu qorundu.

### 56. Dərviş qaşıqları
- Dialoqlar nizamlandı; «yedirdi» → «yedirtdi»; «alan deyil, verən» → «alan yox, verən».

### 57. Dərvişin üst-başı
- Sual-cavab dialoq şəklinə salındı (tire ilə); «səliqə-sahmansız» → «səliqəsiz».
- **Qeyd:** orijinalda ayrıca nəticə yoxdur; aforizm olduğu üçün süni nəticə əlavə edilmədi.

### 58. Dəvəsi oğurlanmış bədəvinin hekayəsi
- Dırnaqsız verilmiş replikalar tire ilə nizamlandı; «yürüyən» → «gedən», «enib» → «düşüb».
- **Qeyd:** son cümlə hekayənin öz ibrəti olduğundan ayrıca nəticə əlavə edilmədi.

### 59. Dibinə darı əkmək
- Dırnaqlar «...» formatına salındı; dialoq tire ilə nizamlandı; deyimin izahı ayırıcıdan sonra saxlanıldı, «Türkiyədə» qeydi ümumiləşdirildi.

### 60. Didro effekti
- **Orfoqrafiya/yazı:** «Əlbətdə» → «Əlbəttə», «Kassasındakl» → «Kassasındakı», «narahlıq» → «narahatlıq», «görsənirdi» → «görünürdü», «iynədən ipəcən» → «iynədən sapa qədər».
- Əsər adı səlisləşdirildi: «Köhnə xələtim üçün peşmanam» → «Köhnə xalatımdan ayrıldığıma peşmanam»; sitat «...» içinə alındı.
- Səpələnmiş çoxnöqtələr nizamlandı; «çar» → «çariça» dəqiqləşdirildi.

## Toplu 5 (hekayə 61–75)

### 61. Dil
- Şeir sətirlərinə vergüllər əlavə edildi; ikinci sətir kiçik hərflə davam etdirildi (bir cümlə axını).

### 62. Dişləri gözləyirəm
- Yazılış: «Mac Donalds» → «McDonald's»; «qallb» → «qalıb», «birinon» → «birinin», «gənç» → «gənc».
- Dialoqlar tire ilə, sitatlar «...» ilə nizamlandı; «ALLAH» → «Allah»; nəticə cümləsi cilalandı.

### 63. Doğru yoldan ayrılmamaq
- **Başlıq orfoqrafiyası:** «AYRİLMAMAQ» → «Ayrılmamaq».
- «tövsiyyəsini» → «tövsiyəsini», «şəkiıldə» → «qədər» (konteksti aydınlaşdıraraq), «rahatlığl» → «rahatlığı»; dırnaqlar «...» formatına salındı; uzun cümlələr yığcamlaşdırıldı.

### 64. Dost istərəm
